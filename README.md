🌍 My Website Project

This is a full-stack web application built using Node.js and Socket.io.

🚀 Features
Real-time communication using Socket.io
Node.js backend server
Static frontend website
API integration for data handling
📁 Project Structure
server/   → Backend (Node.js, Express)
website/  → Frontend (HTML, CSS, JS)
⚙️ Installation
git clone https://github.com/suryagaming145/my-website.git
cd my-website
npm install
▶️ Run Project
Start backend server:
cd server
node index.js
Open frontend:

Just open:

website/index.html

in your browser.
